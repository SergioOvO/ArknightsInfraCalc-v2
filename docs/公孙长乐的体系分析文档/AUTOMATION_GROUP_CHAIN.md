## 1. 一句话结论

自动化组是**发电→赤金产线闭环体系**：利用虚拟电站（承曦格雷伊 ± 森蚺中枢+Lancet-2）堆高有效电站数，驱动温蒂「仿生海龙」清零同行 + 每电站 15% 加成，配合清流跨设施 20%/贸易站，配合第三人（森蚺＞冬时＞异客/掠风），实现赤金线天花板效率。243 典型配置 **140%（清流+温蒂+森蚺）** 至 **130%（清流+温蒂+冬时）**。与迷迭香感知链**完全互补**（各占一条赤金线）。

**与莱茵生命体系是并列关系**，不是子集。

## 3. 成套组成

### 3.1 架构图

```
243 典型配置（温蒂+清流+森蚺 in manu，承曦格雷伊 in power）：
┌─────────────────────────────────────┐
│ 发电站：承曦格雷伊 E2               │
│   → VirtualPower +1                 │
│   → 有效电站 = 3 物理 + 1 虚拟 = 4   │
│                                       │
│ 制造站 manu_3（赤金）：               │
│   温蒂 E2  仿生海龙  15% × 4 = 60%   │
│   清流 E2  每贸易站 20% × 2 = 40%   │
│   森蚺 E2  第三人贡献 ≈ 40%         │
│   ── 合计 140% ──                    │
└─────────────────────────────────────┘
```

### 3.2 成员

| 角色 | 干员 | 设施 | 最低练度 | 作用 |
|------|------|------|---------|------|
| 自动化主机 | 温蒂 | 制造（赤金）| E2 | 仿生海龙：清零同行 + 每站 15% |
| 跨设施 buffer | 清流 | 制造（赤金）| E2 | 每贸易站 +20% |
| 第三人首选 | 森蚺 | 制造（赤金）| E2 | 243/333/153 时最优 |
| 第三人备选 | 冬时 | 制造（赤金）| E2 | 第三人替补 |
| 第三人替补 | 异客/掠风 | 制造（赤金）| E2 | 低配 |
| 虚拟电源 | 承曦格雷伊 | 发电站 | E2 | VirtualPower +1 |
| 虚拟电源（252/342 用）| 森蚺 | 中枢 | E2 | + Lancet-2(0心情) → VirtualPower +2 |
| 虚拟电源挂件 | Lancet-2 | 发电站 | 0 | 0 心情上班，配合中枢森蚺 |

### 3.3 为什么 243 不用「中枢森蚺 + Lancet-2」

理论上：清流+温蒂+冬时 + 中枢森蚺 + Lancet-2(0心情) + 承曦格雷伊 = 6 有效电站 = **160%**。
但代价：
- Lancet-2 占发电位（损失 ~25% 发电 = 12.5% 制造）
- 森蚺占中枢位（损失 凯尔希/Mon3tr 等的 ~8% 制造）
- 总损失 ≈ **20.5%**，净收益 **160% - 140% = +20%，但损失 20.5%**，总体亏的。

所以 243 最优解：**清流+温蒂+森蚺 in 制造，承曦格雷伊 in 发电，森蚺不进中枢**。

### 3.4 252/342 的「中枢森蚺」用法

252/342 只有 2 物理电站，必须靠虚拟电站补足：
- 中枢森蚺 E2 + Lancet-2（0 心情）= VirtualPower +2
- 承曦格雷伊 E2 = VirtualPower +1
- 有效 = 2 物理 + 3 虚拟 = **5**

252 时：清流 40 + 温蒂 15×5 + 队友 = ?
