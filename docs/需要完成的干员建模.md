## 「完全建模」在本项目的含义

| 层级 | 含义 |
|------|------|
| 数据注册 | `operator_instances.json` 有设施绑定 + `skill_table.json` 有 `buff_id` |
| 机制可算 | `atoms` 非空，或 `atoms: []` 且 L2 已落地（如贸易 `gold_flow`） |
| 跨设施闭环 | producer → `GlobalResourcePool` → consumer（P0：木天蓼、宿舍人数链已闭环） |
| 回归锚定 | `verify` / `UNIT_OUTPUT_ANCHORS` 有夹具 |

**非目标**（不会在本仓库补全）：连续心情排班、宿管、宿舍/训练室/加工站求解。办公室和会客室仅开放显式编制的有界静态求值，见 `EFFECT_ATOM_DESIGN.md` §十。

`MECHANICS_REGISTRY.csv` 727 条 ≠ 727 条都要算；当前主力是 **贸易搜索/轮换 + 制造搜索**。
